The Terminal is a command line interface that mimics a classic "Terminal" interface on a computer, engineered to mainly be used by players instead of developers. It takes up the entire screen and has multiple commands, which are contained inside of the Commands ModuleScript inside of ReplicatedStorage.

It depends on the following modules:

| Module | Reason |
| ------ | ------ |
| [CustomEnums](https://devforum.roblox.com/t/1-0-0-customenums-create-your-custom-enum/573300) | Commands API |
| [vLua](https://www.roblox.com/library/4689019964/vLua-5-1) | Raw Code Execution (for invalid commands), `cd` returning |
| [rQuery](https://github.com/EthanMcBloxxer/rQuery) | `music` command |
